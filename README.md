# FullStack Pleace Keanu App

## Create a .env file
```
PORT=4000
REACT_APP_API_URL=http://backend:4000
API=https://placekeanu.com
```

## Run full app
```
docker compose up -d
```

## Run frontend
```
npm run frontend
```

## Run backend
```
npm run backend
```
